# 更新日志

Gleam 的所有重要变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.1] - 2025-12-14

### 新增
- 导出对话到文档：将当前聊天对话导出为思源笔记文档
- 自动图片转换：导出时自动将对话中的 base64 图片保存为文件
- 导出成功提示：导出完成时显示成功提示

## [0.1.0] - 2025-12-14

### 新增
- Gleam 插件初始发布
- 支持 OpenRouter AI 供应商
- 集成到思源笔记停靠面板的聊天界面
- 带搜索功能的模型选择
- 从当前文档注入上下文
- 带代码高亮的 Markdown 渲染
- 多模态支持（图片和音频输入）
- 聊天历史管理
- 消息重新生成
- 参数配置面板（温度、最大令牌数等）
- 复制消息和图片到剪贴板
- 图片缩放和预览
- 带 API 密钥配置的设置面板
- 设置中的默认模型选择
- 最大历史数量配置
- 调试日志开关
- 国际化支持（英语和简体中文）
- 空状态显示
- 加载指示器
- 错误处理和用户反馈

### 功能特性
- **AI 供应商**：OpenRouter 集成
- **聊天界面**：简洁、响应式的聊天 UI
- **上下文注入**：自动包含文档上下文
- **历史管理**：保存、加载和管理对话历史
- **模型选择**：带搜索功能的轻松模型切换
- **参数调优**：自定义 AI 行为
- **多模态**：支持文本、图片和音频
- **Markdown 支持**：完整的 Markdown 渲染
- **用户体验**：直观的控件和反馈

### 技术细节
- 使用 TypeScript 构建
- Webpack 打包
- SCSS 样式
- 模块化架构（管理器、处理器和组件）
- 事件驱动设计
- 本地存储配置和历史
- 流式支持实时响应
